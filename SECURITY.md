# Security Policy

## Supported Versions

As magic-flow doesn't provide an offical release yet, please use the latest version for the security updates.
Current we have two branches to maintain: 
* main branch for magic-flow 2.x
* main-1.x branch for magic-flow 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/magic-flow/security to report the vulnerability you find.
